# CaliTrack — Calisthenics Workout Tracker

A modern bodyweight training app built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Exercise Library** — 21 bodyweight exercises across 5 categories (push, pull, legs, core, full-body) with difficulty levels, muscle targets, and step-by-step instructions
- **Workout Programs** — 7 pre-built routines from beginner to advanced
- **Active Workout Tracker** — Guided workout flow with set tracking, rep counters, hold timers, and rest timers
- **Progress Dashboard** — Stats overview, weekly activity chart, and full workout history
- **Offline-Ready** — All data stored in localStorage; no account or backend needed
